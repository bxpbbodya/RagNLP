# Retrieval-Augmented Generation (RAG) Retrieval-Augmented Generation combines information retrieval with text generation. ## Motivation Large language models may hallucinate or lack up-to-date knowledge. RAG mitigates this by retrieving relevant documents and conditioning generation on them. ## Pipeline 1. User query 2. Retrieval of relevant chunks 3. Optional reranking 4. Generation using retrieved context ## Failure modes RAG can fail if irrelevant documents are retrieved or if the model ignores the context.
